## Introduction
This folder has the following utilities:


### Generate ED25519 keys
- Beckn uses ED25519 keys to sign and verify messages. 
- There are two utilities in this folder that generate this key pair. 
- If you have Go installed on your computer, use `go run generate-ed25519-keys.go` to generate the keys.
- If you do not have Go installed, then use the shell script `./generate-ed25519-keys.sh`.
